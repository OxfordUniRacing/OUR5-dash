## Import project to STM32CubeIDE instructions:

### Prerequsites:
- Download and install **STM32CubeIDE** from https://www.st.com/en/development-tools/stm32cubeide.html (includes ARM GCC compiler, debugger, and ST-Link drivers)

### Steps:
1. Clone this respository with the recursive flag:
```
git clone --recursive https://github.com/OxfordUniRacing/OUR5-dash
```
2. Navigate to STM32CubeIDE/ and open the .project file:
```
double click on .project
```
3. Build the project in STM32CubeIDE:
```
Project => Build all
```
